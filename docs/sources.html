Documentation

Public source inventory

This project uses a combination of structured public datasets and supporting reference documents. The goal is to preserve a clear record of what each source contributes to the analysis.

Public Data Sources

All analysis on this site is based on publicly available City of Chicago data and municipal documentation, supplemented by documented derived fields and bounded interpretation rules.

Core Workforce Data

  • Current Employee Names, Salaries, and Position Titles
    Base workforce dataset used for employee counts, departments, public titles, and compensation fields.
  • Workforce Vacancies
    Used to assess staffing-gap pressure and department-level vacancy concentration.

Classification and Compensation References

  • Classification and Pay Plan materials
    Used to support title normalization, role grouping, and classification crosswalk logic.
  • Job Specifications and related references
    Used as descriptive support for interpreting title groupings and role families.

Governance Context Materials

  • Public governance and hiring-plan materials
    Used as context for role-status interpretation and control-oriented review.
  • Public settlement or role-status reference materials
    Used where relevant to map governance-related categories cautiously.

2022 Structural Evidence Layer

  • Budget and appropriations materials
    Used as department-level staffing and structural context, not as complete org-chart data.
  • Funding, recovery, and transformation materials
    Used as evidence for structural signals related to public-health expansion and modernization.

Documentation Standard

  • Record exact source title, publisher, and date accessed.
  • Note whether the source is a dataset, PDF, memo, or webpage.
  • Document what analytical field or interpretation the source supports.
  • Track manual extraction or inferred logic separately from raw source facts.

Limitation

Some sources used in this project are structured datasets, while others are PDFs or contextual documents. Structural and governance findings are therefore based on documented interpretation rather than a single unified master dataset.
